# 顺序查找（Sequential Search）

顺序查找也称为线形查找，属于无序查找算法。

从数据结构线形表的一端开始，顺序扫描，依次将扫描到的结点关键字与给定值k相比较，若相等则表示查找成功；若扫描结束仍没有找到关键字等于k的结点，表示查找失败。

| 算法名称                      | 类型 | 平均查找长度 | 时间复杂度 |      |
| ----------------------------- | ---- | ------------ | ---------- | ---- |
| 顺序查找（Sequential Search） | 无序 | (n+1)/2      | O(n)       |      |
